活动状态流

每个阶段都能被负责人手动调整，也支持到点自动锁定。
草稿 负责人创建规则，管理员可协作完善候选字段与阶段。
提名中 用户提交候选，Bangumi 自动补全资料，负责人审核。
投票中 按阶段规则限制票数、资格、公开方式，记录基础风控信息。
结算与归档 阶段结束锁票，生成榜单、晋级名单和活动历史页。
